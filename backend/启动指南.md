# 智览平台 — 后端启动指南

## 环境要求

| 服务 | 版本 | 默认地址 |
|------|------|----------|
| Python | 3.10+ | — |
| MySQL | 8.0+ | `localhost:3306` |
| Redis | 7.0+ | `localhost:6379` |
| Elasticsearch | 8.0+ | `http://localhost:9200` |

## 启动步骤

### 1. 配置环境变量

编辑 `.env`，修改数据库地址和 DashScope API Key。

### 2. 安装依赖

```powershell
$env:TMPDIR = "D:\tmp"
pip install -r requirements.txt
```

### 3. 启动 MySQL / Redis / Elasticsearch

确保三个服务均已运行。

### 4. 初始化数据库

```powershell
mysql -u root -p < configs/init.sql
```

### 5. 启动服务

```powershell
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 6. 验证

| 地址 | 说明 |
|------|------|
| http://localhost:8000/health | 健康检查 |
| http://localhost:8000/docs | Swagger API 文档 |
